# PokemonLevelUp Roadmap

PokemonLevelUp is currently focused on release hygiene, naming consistency, and keeping parity with the other RGX sound addons.

## Current Focus

1. Release pipeline stability
- Keep package metadata, TOC files, README, and CurseForge description aligned.
- Maintain tag-driven packaging through GitHub Actions.

2. Addon consistency
- Keep PLU structure aligned with SRLU and FFLU where it makes sense.
- Maintain clean addon naming across repo, package, sound files, and runtime paths.

3. Gameplay polish
- Validate Pokemon-themed sound behavior across supported WoW client versions.
- Keep slash commands and saved settings lightweight and predictable.

## Next Up

1. Discord release notifications
- Set up a dedicated Discord webhook/channel for PokemonLevelUp releases.
- Re-enable and validate Discord release announcements once the channel is configured.

2. Asset polish
- Review logo and media assets for full PLU branding consistency.
- Add any missing branded visuals used by release pages or storefront listings.
