# RGX | Pokemon Level-Up! ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/media/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Pokemon Level-Up!** brings a Pokemon-inspired level-up jingle to World of Warcraft, replacing the default sound with a brighter, nostalgic level-up moment. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/media/kiwi.gif?raw=true) **The Kiwi Says:** "Pika-level-up! Bwwiiiee." *** ## ✨ Features 🎵 Pokemon Sound - A Pokemon-style level-up jingle plays every time you gain a level. 🎚️ Quality Options - Switch between High, Medium, and Low quality variants. 🔇 Smart Muting - Auto-mutes WoW's default level-up sound so you only hear PLU. 💬 Login Message - A branded PLU login message appears on login and can be toggled on or off. 💾 Persistent Settings - Settings are saved between sessions. ⚡ Lightweight - Minimal footprint — no embedded libraries required. 🎮 Multi-Version - Includes support for Retail, Wrath, Burning Crusade, and Classic Era. **Note:** All quality variants are included — no additional downloads required. *** ## 📋 Command Reference The addon can be managed using the following slash commands:
/plu help # Show all available commands
/plu test # Play test sound
/plu enable # Enable the addon
/plu disable # Disable the addon
/plu welcome # Toggle login message
/plu high # Use high quality sound
/plu med # Use medium quality sound (default)
/plu low # Use low quality sound
